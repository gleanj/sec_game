## Perfect Recording Browser Setup:

### Set Your Browser Window Size Exactly:

**Chrome DevTools Method:**
1. Press **F12** (Open DevTools)
2. Press **Ctrl+Shift+M** (Toggle device toolbar)
3. Select "Responsive" at top
4. Enter dimensions: **1080 x 1920**
5. Your terminal will now be EXACTLY 9:16!

**Manual Window Resize:**
1. Set browser to windowed mode (not maximized)
2. Resize to approximately 1080x1920
3. Use OBS to capture just the terminal area
